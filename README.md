# Rock Paper Scissors (C++)

A simple command-line implementation of the classic Rock Paper Scissors game.

## Description

This project was created to reinforce basic C++ concepts such as functions, conditionals, and loops.

The player competes against the computer in a simple game loop.

## How to Play

1. Run the program  
2. Enter `rock`, `paper`, or `scissors`  
3. The computer will randomly choose its move  
4. Both moves and the result (win, lose, or tie) will be displayed  

## Installation

### Option 1: Download ZIP
1. Go to the repository: https://github.com/Izers0/RockPaperScissors  
2. Click the green **Code** button  
3. Select **Download ZIP**  
4. Extract the ZIP file  

### Option 2: Clone with Git
In the terminal, run:

```bash
git clone https://github.com/Izers0/RockPaperScissors.git
```
