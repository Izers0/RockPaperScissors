Rock Paper Scissors
  A Command Line game of Rock Paper Scissors

Description
  A C++ game made to enforce learning of basic C++ concepts using functions, conditions and a game loop.
  The player plays against the computer in a basic game loop.

How to Play
1. Run the program
2. Enter rock, paper or scissors
3. The computer will randomly choose its move
4. Your move, the computers move and the winner or a tie are displayed 

Installation
  Option 1: Download ZIP
      Go to the repository: https://github.com/Izers0/RockPaperScissors
      Click the green Code button
      Click download ZIP
      Extract the ZIP file to a folder on your computer
  
  Option 2: Clone with Git:
      In the terminal, run: git clone https://github.com/Izers0/RockPaperScissors.git

Features
  Randomised computer choices
  Simple Command Line interface - easy to read

Note 
  Only supports single player (user vs computer)
  Two-player support may be added in the future
